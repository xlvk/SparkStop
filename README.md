# nasa-hackathon
A repo made for the nasa space hackahon

## How to run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Go to `localhost:3000` in your browser

OR
1. Clone the repo
2. Run `npm install`
3. Live serve the project
4. Go to `localhost:3000` in your browser (or whatever port you are using)
